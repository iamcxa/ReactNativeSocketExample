# ReactNativeSocketExample

A sample React-Native 0.60 + Node Socket implementation with `react-native-tcp` and `Nodeify`, using `net` as tco socket connector.

## Usage

```bash
# Installation
$ yarn install

# Deploy to device
$ yarn run ios
# or
$ yarn run android

# Start
$ yarn start
```
